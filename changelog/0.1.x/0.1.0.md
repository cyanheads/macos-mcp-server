# 0.1.0 — 2026-05-25

Initial release of macos-mcp-server: macOS system controls via MCP using osascript, screencapture, SwitchAudioSource, and displayplacer.

## Added

- `macos_get_info` — system snapshot: battery, Wi-Fi, hostname, macOS version, uptime, display count
- `macos_check_permissions` — reports Accessibility, Screen Recording, Automation, and Notification status for the calling process
- `macos_manage_apps` — list, launch, quit, force-quit, hide, and show applications
- `macos_manage_windows` — list, focus, move, resize, move_resize, minimize, fullscreen, and close windows
- `macos_control_volume` — get or set system output volume and mute state
- `macos_control_audio` — list audio devices, get current defaults, switch input/output device (requires SwitchAudioSource)
- `macos_control_appearance` — get or set dark/light mode
- `macos_control_system` — lock screen or sleep display
- `macos_take_screenshot` — capture full screen, display, window, or region; saves PNG to disk; optional base64 JPEG preview
- `macos_manage_displays` — list connected displays and apply named layout presets (requires displayplacer)
- `macos_send_notification` — post a notification to Notification Center via osascript
- `macos_manage_focus` — get or set Do Not Disturb / Focus mode
- `macos_manage_finder` — frontmost path, selection, reveal, open_with, and trash
- Resource `macos://system/info` — system snapshot (battery, Wi-Fi, hostname, version, uptime, display count)
- Resource `macos://audio/devices` — all input/output audio devices with current defaults
- Resource `macos://displays` — connected display inventory with current layout
- Services: osascript, system-info, audio, display, screencapture
- STDIO and Streamable HTTP transports via `@cyanheads/mcp-ts-core ^0.9.9`
