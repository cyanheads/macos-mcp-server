- **`.github/FUNDING.yml`** — GitHub Sponsors and Buy Me a Coffee config
- **`funding`** array in `package.json` — GitHub Sponsors and Buy Me a Coffee
- **npm badge** added to README badge row

- **`tsx`** devDependency — no longer needed with `bun run` invocation
